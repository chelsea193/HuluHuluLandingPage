01 — Typefaces

Four font families, four jobs

Loaded from Google Fonts in src/index.css:1 and registered as Tailwind theme tokens / utility classes in the same file.

font-sans · Inter Food Is Energy Latin/English UI: nav links, buttons, tags, chip labels. Also the default <body> font.
font-noto-sans-sc · Noto Sans SC 你多久没有好好吃饭了？ The primary Chinese font site-wide — headlines, paragraphs, card titles, buttons.
font-noto-serif-sc / font-serif · Noto Serif SC 其实，好好吃饭就能解决。 Chinese serif accents — quote headlines, small labels, editorial emphasis.
font-mono · JetBrains Mono COMMON CONCERNS All-caps tracked eyebrows and numeric/code-style tags.
Loaded but unused: .calligraphy-text sets font-family: 'Ma Shan Zheng', cursive; font-size: 28px; in src/index.css:25-29, but no component currently applies the class.
Fallback stack: font-sans → Inter, ui-sans-serif, system-ui, sans-serif · font-serif → Noto Serif SC, Playfair Display, Georgia, serif · font-mono → JetBrains Mono, ui-monospace, SFMono-Regular, monospace.
Note: Tailwind's bare font-serif token resolves to Noto Serif SC first, so plain font-serif and explicit font-noto-serif-sc render identically for Chinese text.

05 — Variant

/version2 (src/v2/)

src/v2/App.tsx is byte-identical to src/App.tsx. Every v2 component uses the exact same font families and size classes as its v1 counterpart — font-noto-sans-sc, font-noto-serif-sc, font-mono, font-sans, font-serif — with only a handful of responsive breakpoint tweaks: e.g. YinYangBalance.tsx shrinks some text-xs labels to text-[10px] sm:text-xs on narrow screens, and the active-food name goes from a fixed text-xl to text-lg sm:text-xl. No new font family or base size is introduced in v2.
